# AI Pong — самообучающийся игровой интеллект

## О проекте

Данный проект представляет собой реализацию классической игры Pong, в которой управление правой ракеткой осуществляет искусственный интеллект, построенный на алгоритме Q-learning. ИИ обучается в реальном времени, анализируя свои действия и получая обратную связь в виде наград или штрафов.

Мне интересна сфера машинного обучения, и я решил на практике разобраться в том, как работают алгоритмы обучения с подкреплением. Так появилась идея сделать самообучающегося игрока в Pong. В процессе разработки я изучил принципы Q-learning, реализовал клиент-серверную архитектуру с Flask и JavaScript, а также добавил визуализацию данных для наглядного отслеживания прогресса модели.

## Технологический стек

- Backend: Python, Flask, scikit-learn
- Frontend: HTML5, CSS3, JavaScript, Chart.js
- ML алгоритм: Q-learning
- Хранение данных: joblib (сериализация модели)

## Архитектура и принцип работы

### Клиент-серверное взаимодействие

Игра разделена на две части:
- Клиентская часть (браузер) отвечает за отрисовку игрового поля, обработку ввода пользователя и визуализацию данных.
- Серверная часть (Flask) содержит Q-таблицу, обрабатывает запросы на предсказание действий и обновляет модель на основе полученных наград.

### Алгоритм Q-learning

Состояние игры описывается тремя параметрами:
- позиция мяча на поле (верх, центр, низ)
- направление движения мяча (вверх или вниз)
- положение ракетки относительно мяча

Возможные действия ИИ: движение вверх, движение вниз, остановка.

Система наград:
- +20 за успешный отскок мяча
- +15 за забитый гол
- -25 за пропущенный гол
- +5 за правильное позиционирование

Параметры обучения:
- Alpha (скорость обучения): 0.15
- Gamma (дисконт будущих наград): 0.92
- Epsilon (коэффициент исследования): начинается с 0.7, постепенно уменьшается до 0.05

### Особенности игровой механики

- Угол отскока мяча зависит от точки касания ракетки
- Каждый удар увеличивает скорость мяча
- Три уровня сложности влияют на скорость реакции и перемещения ИИ
- Режим авто-обучения позволяет проводить тысячи игр без участия человека

## Установка и запуск

### Требования

- Python 3.8 или выше
- Современный веб-браузер (Chrome, Firefox, Edge)

### Инструкция по установке

1. Клонировать репозиторий

git clone https://github.com/username/ai_pong_ml.git
cd ai_pong_ml

2. Создать виртуальное окружение

python -m venv venv

3. Активировать виртуальное окружение

Windows:
venv\Scripts\activate

macOS / Linux:
source venv/bin/activate

4. Установить зависимости

pip install flask scikit-learn numpy pandas joblib

5. Запустить приложение

python app.py

6. Открыть в браузере

http://127.0.0.1:5000

## Управление

- Левая ракетка управляется движением мыши по игровому полю
- Кнопка Старт запускает игру с трёхсекундным обратным отсчётом
- Кнопка Пауза приостанавливает игровой процесс
- Кнопка Сброс игры обнуляет счёт и возвращает мяч в центр

## Настройки

### Выбор сложности

- Лёгкий: низкая скорость реакции ИИ и уменьшенная максимальная скорость мяча
- Средний: сбалансированные параметры для комфортной игры
- Сложный: быстрая реакция ИИ и высокая скорость мяча

### Управление моделью

- Сохранить: экспорт текущей Q-таблицы в файл
- Загрузить: импорт ранее сохранённой модели
- Сбросить: очистка Q-таблицы и начало обучения с нуля

### Авто-обучение

- 100 игр: быстрая тренировка
- 500 игр: средняя тренировка
- 1000 игр: полная тренировка

В режиме авто-обучения обе ракетки управляются ИИ, что позволяет модели накапливать опыт без участия игрока.

## Визуализация обучения

- График обучения отображает три кривых:
  * Epsilon (оранжевый) — снижение исследования со временем
  * Q-таблица (зелёный) — рост количества изученных состояний
  * Скорость изменения epsilon (синий) — динамика обучения

- Q-таблица показывает ценность каждого действия для всех встреченных состояний. Лучшие действия выделены цветом.

## Заключение

Проект позволил на практике реализовать алгоритм Q-learning, интегрировать его в игровое приложение и наблюдать процесс обучения в реальном времени. Полученный опыт может быть использован для дальнейшего изучения методов обучения с подкреплением и их применения в других задачах.
